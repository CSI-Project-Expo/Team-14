Decision simulator UI components

### Supabase Configuration (Local / Demo Only)

This prototype uses Supabase for storing simulation data.

To run locally:
1. Create a Supabase project
2. Replace `SUPABASE_PROJECT_URL` and `SUPABASE_ANON_KEY` in `script.js`
3. Do not commit actual keys to GitHub

Keys are intentionally excluded for security reasons.
